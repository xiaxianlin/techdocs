# 基本公式

- <漫反射光颜色> = <入射光颜色> x <表面基底色> x cos θ
- <漫反射光颜色> = <入射光颜色> x <表面基底色> x (<光线方向> • <法线方向>)
- <环境反射光颜色> = <入射光颜色> x <表面基底色>
- <表面反射光颜色> = <漫反射光颜色> + <环境反射光颜色>
- <雾化因子> = (<终点> - <当前点与视点的距离>) / (<终点> - <起点>)
	- 其中：<起点> ≤ <当前点与视点的距离> ≤ <终点>
- <片元颜色> =  <物体表面颜色> x <雾化因子> + <雾的颜色> x (1 - <雾化因子>)
