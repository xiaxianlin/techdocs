# 基础

### 概念

| SQL概念     | MongoDB概念 | 说明                       |
| ----------- | ----------- | -------------------------- |
| database    | database    | 数据库                     |
| table       | collection  | 数据表/集合                |
| row         | document    | 数据行/文档                |
| column      | field       | 数据列/字段                |
| index       | index       | 索引                       |
| table joins | -           | 表连接，Mongo不支持        |
| primary key | primay key  | 主键，Mongo自带_id主键字段 |
